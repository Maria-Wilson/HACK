# Campus Blocks & Route Navigation

A simple web application to help navigate through various campus blocks and route points.

## Features
- **Block Overview**: Visual representation of Block A, B, C, and D.
- **Route Points**: Clear listing of key navigation points from the Main Gate to the Parking Area.
- **Interactive Navigation**: Easy transition between navigation sections.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript

## Getting Started
To view the project locally:
1. Clone the repository.
2. Open `index.html` in your favorite web browser.

## File Structure
- `index.html`: Main landing page with block overview.
- `route.html`: Detailed route navigation page.
- `Index1.html`: Alternative/backup index page.
- `*.jpeg`/`*.jpg`: Campus images.
